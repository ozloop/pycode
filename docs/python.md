

## scrapy

http://www.scrapyd.cn/doc/160.html

1. install
     conda install -c conda-forge scrapy 
2. create project 
    scrapy startproject scrapy001
3. write and run spider
    scrapy crawl  cblog
4. debug tool   
    scrapy shell http://lab.scrapyd.cn
